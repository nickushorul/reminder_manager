# Reminder Manager pentru Home Assistant

Un sistem personal de remindere nelimitate, gestionate complet din interfata Home Assistant, fara YAML pentru utilizare zilnica.

## Instalare prin HACS
1. Adauga acest repository la HACS (Custom Repositories -> Integration).
2. Descarca integrarea.
3. Restarteaza Home Assistant.
4. Adauga integrarea din Configuration -> Integrations.
